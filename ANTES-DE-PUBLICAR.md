# Confirma — versión de revisión

Esta carpeta contiene la primera versión del rediseño público.

## Ver localmente

Abrí `index.html` con Chrome, Edge o Firefox.

## Publicar

Subí el contenido de esta carpeta a la raíz del repositorio de GitHub Pages, conservando `CNAME`.

## Seguridad

La herramienta interna `gestion.html` y sus scripts de Supabase fueron excluidos intencionalmente. GitHub Pages es público y no debe utilizarse para alojar una herramienta interna que consulta información de clientes sin una capa de autenticación y políticas RLS verificadas.

## Antes de la publicación definitiva

- Revisar el texto comercial y los nombres de cada servicio.
- Confirmar los cursos que deben permanecer visibles.
- Verificar el formulario de contacto de Formspree.
- Incorporar el enlace de pago definitivo antes de habilitar “Comprar ahora”.
